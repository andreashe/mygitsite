Beschreibung: so wird's gemacht

In diese Datei ist links von diesem Text eine Shockwave-Datei (Macromedia) eingebettet. Voraussetzung ist, daß der WWW-Browser Shockwave-Dateien (*.dcr) anzeigen kann.
.

Wenn Sie ein Plugin, haben, zum Beispiel für Netscape, wird die Datei direkt im Anzeigefenster des Netscape-Browsers angezeigt. Im WWW können Sie das  Shockwave-Plugin für Netscape downloaden 

Wenn bei Ihnen nichts angezeigt wird, kann das daran liegen, daß Ihr WWW-Browser das <object>-Tag noch nicht oder noch nicht vollständig interpretiert. Probieren Sie in diesem Fall das  Einbinden von Multimedia nach herkömmlicher Netscape-Syntax
