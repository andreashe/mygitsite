Browser-interne Verweise definieren

Die folgenden Spezial-Referenzen werden nur von Netscape interpretiert. Es handelt sich zum Teil um nützliche, zum Teil um weniger nützliche Verweise einige Verweise führen zu internen, in der Browser-Software gespeicherten Zielen, andere zu Zielen im WWW.

Verweise zu softtware-internen Zielen

 javascript: - Dieser Verweis ruft eine interne Browser-Seite von Netscape auf, in der man JavaScripts Zeile für Zeile interpretieren und auf Richtigkeit prüfen lassen kann. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="javascript:">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 livescript: - Dieser Verweis ruft eine interne Browser-Seite von Netscape auf, in der man LiveScripts (Vorgänger-Version von JavaScript) Zeile für Zeile interpretieren und auf Richtigkeit prüfen lassen kann. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="livescript:">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 mocha: - Dieser Verweis ruft eine interne Browser-Seite von Netscape auf, in der man Mocha-Code Zeile für Zeile interpretieren und auf Richtigkeit prüfen lassen kann. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="mocha:">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 about: - Dieser Verweis ruft die interne Browser-Seite mit Versionshinweisen auf (bei Netscape 4.0 nicht mehr - dort führt dieser Befehl zu einem Ziel im WWW). Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="about:">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 about:mozilla - Dieser Verweis ruft eine nicht ganz so ernstzunehmende, interne Browser-Seite von Netscape auf. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="about:mozilla">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 about:logo - Dieser Verweis ruft eine interne Browser-Seite auf, die das Netscape-Logo in voller Größe darstellt. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="about:logo">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 about:plugins - Dieser Verweis ruft eine interne Browser-Seite auf, in der angezeigt wird, welche Netscape-Plugins (z.B. Live-Chat) beim Anwender installiert sind. Wenn keine Pugins installiert sind, bleibt die Seite leer. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="about:plugins">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 about:license - Dieser Verweis erzeugt eine interne Browser-Seite, in der die Lizenzvereinbarungen angezeigt werden, die beim Erstaufruf von Netscape einmalig erscheinen. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="about:license">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)

 about:rsalogo - Dieser Verweis ruft eine interne Browser-Seite auf, die das RSA-Logo für den von Netscape unterstützten Security-Standard darstellt. Um einen solchen Verweis zu setzen, notieren Sie folgenden Befehl:
<a href="about:rsalogo">Verweistext</a>
(wobei Sie als "Verweistext" notieren können, was Sie wollen)
