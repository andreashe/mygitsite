Script-Bereiche definieren

Im Kopf einer HTML-Datei werden häufig Bereiche für Script-Sprachen definiert. Die bekannteste und bedeutendste Script-Sprache ist JavaScript. Allgemeine Informationen zu JavaScript siehe  JavaScript und HTML. Nähere Information zu Script-Bereichen für JavaScript siehe  JavaScript-Bereiche definieren.

Beispiel:

<head>
<script language="JavaScript">
   ... JavaScript-Befehle ...
</script>
... andere Angaben im Dateikopf ...
</head>

Erläuterung:

Mit <script language= .... > definieren Sie einen Script-Bereich. Hinter language= geben Sie die Script-Sprache an. Die Angabe sollte in Anführungszeichen stehen.

Style-Sheets definieren

Im Kopf einer HTML-Datei werden Bereiche für Style-Sheets definiert. Style-Sheets sind definierte Formatvorlagen für Absatztypen, wie man sie aus Textverarbeitungs- und DTP-Programmen kennt.

Beispiel:

<head>
<style type="text/css">
... Style-Sheet-Definitionen ...
</style>
... andere Angaben im Dateikopf ...
</head>

Erläuterung:

Mit <style type= .... > definieren Sie einen Definitionsbereich für Style-Sheets. Hinter type= geben Sie den Typ der verwendeten Style-Sheet-Syntax an. Die Angabe sollte in Anführungszeichen stehen. Weitere Informationen siehe  Style-Sheets definieren im Abschnitt über  Style-Sheets.

Hintergrundmusik definieren


Anzeigebeispiel: So sieht's aus (bzw. hört sich's an)

Sie können bestimmen, daß beim Aufruf einer HTML-Datei eine Hintergrundmusik ertönt. Dazu gibt es derzeit zwei Lösungen. Beide der im folgenden beschriebenen Lösungen - die eine für Microsoft's Internet Explorer, die andere für Netscape - sind proprietär. Ärgerlich ist vor allem, daß man immer beide Befehle braucht, um das gewünschte Ziel zu erreichen.

Beispiel:

<head>
<bgsound src="datei.mid" loop=infinite> <!-- Microsoft -->
</head>
<body>
<embed src="datei.mid" width=0 height=0 loop=true autostart=true> <!-- Netscape -->

Erläuterung:

Mit <bgsound ...> definieren Sie im Kopf einer HTML-Datei eine Hintergrundmusik nach Microsoft-Syntax für die Anzeige der Datei (bgsound = background sound = Hintergrundmusik). Mit <embed ...> erreichen Sie das gleiche für Netscape (embed = einbetten). hinter der Angabe src= folgt in beiden Fällen die Angabe der gewünschten Musikdatei (src = source = Quelle). Es sollte sich möglichst um Dateien der Typen MID, AU oder WAV handeln. Um eine sichtbare Anzeige des Abspiel-Players zu unterdrücken, sind bei der Netscape-Syntax die Angaben width=0 height=0 erforderlich (width = Breite, height = Höhe). Auch den automatischen Start des Abspielvorgangs müssen Sie in der Netscape-Syntax explizit angeben - durch autostart=true. Schließlich können Sie noch bestimmen, ob die Musikdatei nur einmal, mehrmals oder endlos oft (bis zum Aufrufen einer anderen HTML-Datei) abgespielt wird. Nach der Microsoft-Syntax können Sie mit loop=infinite eine Endloswiederholung erzwingen. Wenn Sie die Wiederholungszahl begrenzen wollen, geben Sie anstelle von infinite einfach die gewünschte Anzahl der Wiederholungen an. Bei der Netscape-Syntax haben Sie nur die Wahl zwischen Endloswiederholung und keiner Wiederholung. Um eine Endloswiederholung zu erzwingen, notieren Sie die Angabe loop=true. Ansonsten lassen Sie die Angabe einfach weg.

Beachten Sie:

Die Microsoft-Syntax muß im Dateikopf der HTML-Datei stehen. Die Netscape-Syntax sollte innerhalb des Dateikörpers stehen (siehe auch  Grundgerüst einer HTML-Datei).

Im obigen Beispiel wird vorausgesetzt, daß sich die Musikdatei im gleichen Verzeichnis befindet wie die HTML-Datei. Wenn die Datei in einem anderen Verzeichnis steht, müssen Sie den relativen oder absoluten Pfadnamen angeben. Das funktioniert genau so wie beim  Einbinden von Grafiken.

Die Ausgabe einer Hintergrundmusik setzt beim Anwender natürlich entsprechende Hardware (Soundkarte, Lautsprecher) voraus. Ferner muß der WWW-Browser die Möglichkeit haben, die Ausgabe der Musikdatei zu steuern - zum Beispiel mit Hilfe eines  Plugins.

Näheres zur Netscape-Syntax für Multimedia siehe  Multimedia einbinden (Netscape-Syntax).
