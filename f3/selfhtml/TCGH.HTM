Browser-interne Grafik-Ressourcen referenzieren

Die folgenden Spezial-Referenzen werden nur von bestimmten WWW-Browsern interpretiert. Es handelt sich zum Teil um nützliche, zum Teil um weniger nützliche Grafiken.

Der Vorteil ist, daß Sie keine separaten Grafikdateien dafür brauchen. So können Sie zum Beispiel eine einzelne HTML-Datei weitergeben, die trotzdem Grafiken enthält - sofern es sich um eine browser-interne Grafik-Ressource handelt.

Notieren Sie die Befehle zum Referenzieren von internen Grafik-Ressourcen an der Stelle, wo sie die Grafik wünschen. Diese Grafiken können Sie genau so  beschriften oder  mit Text umfließen lassen wie normale Grafiken.
