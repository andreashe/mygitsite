Allgemeines zur Verwendung

Über das Objekt navigator können Sie in einem JavaScript Informationen darüber ermitteln, welchen WWW-Browser der Anwender verwendet sowie einige nähere Spezifikationen dazu.

Instanz des Objekts navigator erzeugen

Eine Instanz von navigator brauchen Sie nicht eigens erzeugen. Sie können Eigenschaften von navigator direkt ansprechen.

Schema:

navigator.Eigenschaft

Eigenschaften

navigator.appCodeName
Lesen

Ermittelt den Codenamen des vom Anwender benutzten WWW-Browsers, bei Netscape z.B. den Namen Mozilla.

navigator.appName
Lesen
Ermittelt den offiziellen Anwendungsnamen des vom Anwender benutzten WWW-Browsers, bei Netscape z.B. den Namen Netscape.

navigator.appVersion
Lesen
Ermittelt die Version und die Betriebssystemplattform des vom Anwender benutzten WWW-Browsers, bei Netscape 2.0 unter Windows 95 beispielsweise das Ergebnis 20. (Win95, I). Der letzte Buchstabe kann I für International Release oder U für U.S. Release sein.

navigator.userAgent
Lesen
Ermittelt die Daten des vom Anwender benutzten WWW-Browsers genau so, wie sie im HTTP-Protokoll an den aufgerufenen WWW-Server übermittelt werden. Beispiel: Mozilla/2.0 (Win16; I).

Methoden

Das Objekt navigator hat keine eigenen Methoden.
