Tips & Tricks

Die folgenden Hinweise wollen helfen, das vorliegende Dokument optimal zu nutzen.
  • Falls Sie dieses Dokument gerade online lesen und für Ihre Online-Verbindung Zeit- oder Volumengebühren zahlen: laden Sie sich das Dokument besser auf Ihren Rechner. Sie können dieses Dokument problemlos  downloaden. Nach dem Download haben Sie eine ZIP-Datei, die Sie mit einem geeigneten UNZIP-Programm entpacken können (unter MS Windows z.B mit  WinZip ). Entpacken Sie die ZIP-Datei in ein eigens dafür eingerichtetes Verzeichnis auf Ihrer Festplatte. So können Sie das Dokument ohne Streß lesen.

  • Während Sie an eigenen HTML-Dateien arbeiten, sollten Sie immer ein oder mehrere WWW-Browser im Hintergrund geöffnet haben, in denen die editierte Datei angezeigt wird. Moderne WWW-Browser erlauben mehrere "Instanzen". Auf diese Weise können Sie in einer Instanz des Browsers Ihre Datei anzeigen, in einer anderen Instanz "HTML-Dateien selbst erstellen". So können Sie bequem zwischen Dokumentation, Editieren der eigenen Datei und Anzeige der eigenen Datei hin- und herwechseln. In Netscape (englische Version) erzeugen Sie eine neue Browser-Instanz beispielsweise mit dem Menübefehl "File/New Web Browser".

  • Kopieren Sie sich Beispiele zu Codes, die Sie verwenden wollen, einfach über die Datenaustausch-Schnittstelle (Zwischenablage) Ihres Rechners direkt aus diesem Dokument in Ihren Editor. Dann brauchen Sie den Code nur noch anzupassen und vermeiden entscheidende Tippfehler. Dies gilt jedoch nur, wenn Sie HTML-Dateien direkt, also mit einem Texteditor oder einem textorientierten HTML-Editor bearbeiten, nicht bei Verwendung eines Wysiwyg-Editors (siehe auch  HTML-Editoren).
