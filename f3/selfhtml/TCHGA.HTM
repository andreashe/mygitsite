Grafischen Buttons zum Absenden definieren


Beschreibung: so wird's gemacht

Wenn bei Ihnen JavaScript funktioniert, hat ein Klick auf die folgende Grafik die gleiche Bedeutung wie der Back-Button im WWW-Browser. Das wird dadurch erreicht, daß beim Absenden des Formulars (beim Klicken auf die Grafik) mit der JavaScript-Befehl onSubmit="history.back()" im einleitenden <form>-Tag ausgeführt wird.
