Hintergrundfarbe für Layer


Anzeigebeispiel: So sieht's aus

Sie können für jeden definierten fest positionierten oder fließenden Layer eine eigene Hintergrundfarbe definieren.

Beispiel 1:

<layer name="Beispiel" left=300 top=250 width=150 bgcolor=#FFFFCC>
<h1>Eine &Uuml;berschrift</h1>
<img src="datei.gif" alt="Ein Bild">
</layer>

Beispiel 2:

<ilayer name="Beispiel" width=400 bgcolor=#000066>
<h1><font color=#FFFFFF>Eine &Uuml;berschrift</font></h1>
<img src="datei.gif" alt="Ein Bild">
</ilayer>

Erläuterung:

Mit der Angabe bgcolor= im einleitenden Tag von <layer...> oder <ilayer...> bestimmen Sie die Hintergrundfarbe (bgcolor = background color = Hintergrundfarbe). Dabei gelten die Regeln zum  Definieren von Farben in HTML.

Beachten Sie:

Wenn Sie eine Hintergrundfarbe für einen Layer definieren, sollten Sie für Text, der in den Layern erscheint, eine kontrastierende Schriftfarbe wählen. Dies kann zum Beispiel wie im zweiten der obigen Beispiele durch <font color=...>...</font> geschehen (siehe auch Abschnitt über  Schriftfarben).

Hintergrundbild (Wallpaper) für Layer


Anzeigebeispiel: So sieht's aus

Sie können für ein für jeden definierten fest positionierten oder fließenden Layer ein eigenes Hintergrundmuster (Wallpaper) definieren. Das funktioniert genauso wie bei  Hintergrundbildern (Wallpapers) für den gesamten Dateikörper.

Beispiel 1:

<layer name="Beispiel" left=300 top=250 width=150 background="datei.gif">
<h1>Eine &Uuml;berschrift</h1>
<img src="datei.gif" alt="Ein Bild">
</layer>

Beispiel 2:

<ilayer name="Beispiel" width=400 background="datei.gif">
<h1><font color=#FFFFFF>Eine &Uuml;berschrift</font></h1>
<img src="datei.gif" alt="Ein Bild">
</ilayer>

Erläuterung:

Durch die Angabe background= im einleitenden Layer-Tag können Sie ein Hintergrundbild für den Layer bestimmen (background = Hintergrund).

Beachten Sie:

In den obigen Beispielen wird vorausgesetzt, daß sich die Grafikdatei im gleichen Verzeichnis befindet wie die HTML-Datei. Wenn die Grafik in einem anderen Verzeichnis steht, müssen Sie den relativen oder absoluten Pfadnamen angeben. Das funktioniert genau so wie beim  Einbinden von Grafiken.

Inerhalb dieses Dokuments finden Sie im Abschnitt über typische Grafiksorten für WWW-Seiten ein paar  typische Hintergrundbilder.
