Beschreibung: so wird's gemacht

Die folgende eingebettete Datei ist eine MIDI-Datei. Das ist ein Format für Musik, die mit Synthesizern über eine entsprechende PC-Schnittstelle direkt in solche Dateien gespielt und nachbearbeitet werden kann. MIDI-Dateien haben den Vorteil, daß sie im Verhältnis zu der Länge der Musik vergleichsweise klein sind.

Die Anzeige wurde auf eine Breite von 200 und eine Höhe von 55 gesetzt. Das ist die Anzeigebreite von Crescendo, einem der beliebtesten MIDI-Plugins für Netscape. Im Internet können Sie das  Crescendo MIDI-Plugin downloaden .

Voraussetzung zum Genuß ist weiterhin eine entsprechende Hardware (Soundkarte, Lautsprecher).

Das dargebotene Musikstück ist ein Ragtime von  Ron Abruzese (72144.1341@compuserve.com)
