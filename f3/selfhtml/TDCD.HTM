Allgemeines zur Verwendung

Eine Objektinstanz von FENSTERNAME.document.anchors wird automatisch erzeugt, wenn der WWW-Browser einen  Anker für Verweisziele in der HTML-Datei vorfindet.

Eigenschaften

FENSTERNAME.document.anchors.length
Lesen
Ermittelt die Anzahl der in einer HTML-Datei definierten Anker. So können Sie etwa mit var Anker = window.document.anchors.length; die Anzahl der in der Datei definierten Anker in einer Variablen speichern und diese weiterverarbeiten.

Methoden

FENSTERNAME.document.anchors hat keine eigenen Methoden.
